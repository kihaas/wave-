# Wave

**Wave** is an early-stage web application built using the Flask framework, focusing on frontend development with HTML, CSS, and JavaScript. The application serves as a simple messenger, featuring user registration, login, a main page, and a chat interface.

---

## ğŸš€ Project Overview

Wave is designed to provide a basic yet functional messaging platform. Users can:

- **Register** and **login** to access the platform.
- **Send and receive messages** in real-time.
- **Navigate** through a clean and responsive user interface.

The backend, built with Flask, handles user authentication and message routing, while the frontend emphasizes a smooth and intuitive user experience.
